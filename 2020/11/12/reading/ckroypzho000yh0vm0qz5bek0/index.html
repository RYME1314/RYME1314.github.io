《游戏剧本怎么写》笔记

2020-11-12 reading

1.主题与故事

1.1游戏剧本创作流程

游戏剧本的创作过程包括两个阶段：分别是策划阶段与剧本制作阶段

策划阶段:确定方向和内容，确定游戏的根本与根基；

剧本制作阶段:遵循策划内容创作，也因此在策划阶段之后

实际上，在实际创作剧本时，“故事”、“角色”、“世界”、“主题”这四个主要部分是同时创作完成的，思维是在这四个成分之间来回跳跃的，它们共同影响最终剧本的完成

1.2主题

一个游戏在立项的时候就应有主题确定出来。无论主题或长或短，都应该有明确的方向，即主题确定整个游戏的方向。

一般来说，在方向感的基础上，主题短句则用词要抽象，长句则要给人吸引力

1.3故事

主题确定方向，进而确定故事是什么样子的
而故事需要传达游戏的主题，以及主角的目的是什么

设定主题，一般要注意以下几点，接下来结合某一项目进行简单说明:

　　1. 游戏目的与主题联系:目的物引发主题（在该项目中游戏的主题是反思，主角的目的是复兴门派，目的与主体之间应该形成这样的循环，即门派覆灭引发主角的反思从而使主角走上复兴门派的道路）；
　　2. 主题足以创作故事:扩充主题，主题与目的拼合

1.4主题与故事

为了使主题与故事之间相符合，应该从主题中导出故事。

方法:从主题引出故事，从主题中发散思维，挖掘故事发展的原因，引出整个故事
材料（当没有头绪时不妨查查资料）:

　　1. 素材库:话题库、创意库记录创意与主题
　　2. 借鉴“原型”
　　3. 使用角色和世界设定，角色创作故事，世界创作故事

1.5表现主题

关于主题的表现形式，需要注意的是“主题不能写在台词里”，应该依靠角色选择与角色行为来表现主题，以故事发展来表现主题

2.角色

2.1角色

角色是形象与性质的总体
角色需要推动故事发展

编剧之能在于“能否着重表现角色的目的与意志”（如对师父师叔的回忆，对门派其他人的态度，对师叔的态度，对敌方势力的态度，以及主角与人、人与人之间、主角内心的纠葛），在于“如何表现角色向目标前进”（主角深入敌巢、找回秘宝、带领大家复兴门派）。前者与后者的区别在于，前者主要是内部情感、态度来表现，后者以外部的行为来表现。

纠葛，强化角色之间的纠葛（即人与人的对立，或内心的纠结），是一种体现角色意志、目的、和感情的手段。纠葛包括战斗对立冲突等“外在纠葛”与内心的“内在纠葛”

体现角色的魅力与原则，能够让玩家产生共鸣

创作角色三个重点：

  1. 角色有魅力有个性（角色设定问题）

  2. 在不同角色定位上多花心思，避免角色重复（角色定位问题）

  3. 角色的目的意志感情和行动与故事相联系（角色与故事关联性问题）

    接下来将从角色的设定、定位、关联性这几方面进行总结。

2.2角色设定

角色设定表是帮助设计角色形象的重要工具，其中主要涉及以下几个项目：
名字（名字决定角色印象，其中主要的因素是发音，名字中有g或d发音的角色往往给人坏人印象，l对应诱惑或怪异，n对应温柔可爱，p有些俏皮，t更像是主角；但我认为这只在国外或者音译作品中有体现，在国内个人更倾向于主角名字的易读性）、年龄（能够表明角色之间的关系，年龄本身能让玩家联系到某种印象）、职业（职业带有明显的先入为主的印象，或者可以制造反差）、相貌（主要写与故事相关的特征）、头发（体现角色内在性格、外在个性，或者状态变化）、体格（体格反应个人年龄、状态、生活习惯与性格）、服装（帮助把握角色详细信息，如兴趣和思维方式）、性格（能够表示角色情感或行为的倾向，考虑性格的成因，让台词更有深度更贴合角色）、特技（凸显不同的特性，可以用以实现目标的有效手段，不一定要是攻击技能）、弱点（让玩家更容易代入角色，唤起共鸣，人物鲜活）、口吻（直接体现角色个性，增加口头禅）、剧本设定（角色目的、经历、状况以及必要信息等）

在性格的设计中，“定位型性格”是一个比较受重视的手法，所谓“定位型性格”即游戏中常见“哥哥那样的性格”这之类的说法。定位型性格的用处在于：

  1. 把身份投射到角色性格上，使人际关系更加直观

  2. 定位型性格的变化阐明角色立场及变化

  3. 利用定位型性格让角色设定更有说服力

    值得注意的是，因为定位型性格中有刻板印象的作用，在实际应用的过程中，应该避免刻板、尝试增加错位反差的设定或性格

    另外，角色设定表要准确传达信息，避免想当然

3.角色定位

3.1角色的定位

定位是分配给故事中各个角色的立场，可以明确角色之间关系，清楚掌握角色行动，避免角色重复
外观相同、定位不同的角色之间，有明显区别；但外观不同，定位相同的角色之间会因为外观特征而有印象的顶替。

3.2角色的5种定位：

  1. 与游戏系统直接关联的定位：
    1.1 主人公
    1.2 待攻克对象
    1.3 系统角色

  2. 与主人公对立的定位：（推动剧本发展）
    2.1 敌人角色
    2.1.1 恶人角色
    2.1.2 敌对角色
    2.2 难关角色
    2.3 竞争者角色

  3. 与主人公同阵营的定位：（推动剧本发展）
    3.1 情侣角色

    3.2 伙伴角色

    3.3 家人角色

    1. 给故事带来变化的定位：（辅助剧本发展）

    2. 1 契机角色

    3. 2 救星角色

    4. 3 叛徒角色

    5. 4 贤者角色

    6. 补充和强化故事的定位：（辅助剧本发展）
      5.1 缓冲角色
      5.2 深论角色
      5.3 动物角色
      5.3.1 “妹妹”角色
      5.4 闲杂角色
      一个角色可以拥有多个定位，角色越重要，拥有定位越多
      且故事发展可推动定位改变

      接下来对这几种分类分别说明。

3.2.1与游戏系统直接关联的定位

1)主人公

主人公作为玩家的分身，是玩家容易带入感情的角色

为了实现这一点，主人公的描绘方式应尽量使玩家更容易带入主角的角色，在于：

  1. 主人公的（外观）形象要以没有太多特征的方式描绘，感觉不到个性

  2. 允许玩家自定义主人公的名字，增加可选的余地，让主人公角色形象更贴近玩家

  3. 尽量不让主人公有过多台词

  4. 不能强制玩家接受主人公的个性，给以容易让大多人接受的个性，如优柔寡断

换位型主人公与角色扮演型主人公，前者如《勇者斗恶龙》系列以及多数文字冒险游戏（AVG）一般，让主人公角色更贴近玩家，是遵循以上朴素的描绘方式的，即让玩家去想“我想这样做”；后者如《最终幻想》、《三国无双》等作品一般，需要玩家积极贴近主人公，即让玩家想“我想变成这样”。

设计角色扮演型主人公时应该注意以下要点：

  1. 避免玩家失去玩游戏的激情，避免主人公采取预想之外的行动，或者有玩家无法接受的意志与感情；
  2. 另外在角色设定上，完美无缺的性格或者外貌以及难以接受的性格也会令玩家失去热情。

    2)待攻克对象

    待攻克对象往往体现游戏的目的，其中既包含游戏最终目的，也包含章节的阶段目的。

待攻克对象的设置应适度，可以为游戏带来紧张感和乐趣；但如果过多的待攻克对象出现，会使得玩家感觉混乱。应保证每个关键部分待攻克对象的数量不会过多，给予适当的删减与时间线错位。

重要的是让玩家时刻清楚谁才是当前的待攻克对象。

待攻克对象的魅力与描写细致效果会影响玩家攻克欲望。

3)系统角色

系统角色是负责让玩家与游戏系统进行交互的角色，比如存档角色与商贩等等。
不会给系统角色安排重要的定位，主要目的只在于发挥游戏系统的功能。
引导员是为玩家介绍游戏系统，提供推进游戏的提示

系统角色的特殊用法：
有时候我们也会让系统角色加入故事中，发挥其他定位的作用，能够使故事发生戏剧性变化。但这种手法不宜滥用，滥用会使玩家处于长期紧张状态，也会让玩家适应这一变化无法发挥应有的戏剧性效果。

3.2.2与主人公对立的定位

让角色对立便于创作故事，推动故事发展

1)敌人角色

敌人角色与主人公敌对，妨碍主人公达成目的
敌人角色可以与主人公敌对、威胁主人公、挫败主人公达到目的、妨碍主人公达到目的，也可以与主人公有相同目的而形成竞争，也可以对主人公怀恨在心、意图消灭主人公。

设置敌人角色的作用与重要性：

  1. 作用在于强化主人公达成目的的意志，燃起玩家消灭敌人的热情；

  2. 决定故事是否有趣的关键在于如何在达成目的的过程中安排敌人，如何生动有趣地描写这一过程（可以设置一些阻碍主人公的敌人，并给出获得伙伴以及获得成长的机会，对成长的过程和打败敌人的高潮部分进行渲染）；

  3. 敌人角色的描绘方法会影响到主人公给人的形象，敌人角色与主人公之间形成直接比较，人们会对比二者的强弱，如果敌人角色被设计成十分强大的角色，那么会使得战胜它的主人公更强大，强化玩家的动机。

敌人的角色还包括恶人角色和敌对角色两类：

  1. 恶人角色体现“绝对的恶”，即绝对的坏人，有恶行出现

    并且恶人角色的外观应该是可怖的。同时也可以制作角色（外观）设定与角色定位之间的反差，带来意外性的角色设定

    其中善恶的基准既要以现实世界的普遍价值观为准，也要以游戏内的世界观以及角色设定进行补充强化

  2. 敌对角色通过与主人公采取敌对的行为实现敌人角色的定位，是游戏的斗争性、竞技性使敌对角色成为敌人

    游戏系统能够多大程度上将战争与竞技融入游戏并体现出趣味性，影响玩家对敌对角色的认可程度（是指认可敌对角色符合这一定位）

    如何使敌对角色看起来更“恶”？

    1. 让敌对角色和恶人角色一样有可怖的外观，这是最直观最简单的方法

    2. 通过剧本，为敌对角色安排恶事，如违反规则或者违背道德准则的行为

      敌对角色和恶人角色之间的区别：
      恶人角色原本就是恶人所以做出恶事，而敌对角色是做出恶事所以是恶人。为了方便理解，应该是在游戏主人公行动的时间内定义的，恶人角色是在主人公之前就有恶行并被定义为恶人，而敌对角色是主人公出现后因为恶行而被定义为恶人的。总之，敌对角色和恶人角色在最初的设计中都是敌人角色。

      2)难关角色

难关角色拦住主人公去路，阻止故事发展，测试主人公实力
难关角色的作用:

  1. 测试主人公的实力
  2. 赐予主人公新的能力与物品，帮助主人公成长
  3. 让故事产生起伏
  4. 促进玩家对主人公产生情感代入
    让难关角色（或者敌人角色）被击败后成为伙伴：
    在被击败后即完成了身为敌人的作用，为了避免该角色处于挂起状态，需要重新赋予其定位，否则只能通过死亡、失踪等方式抹去

    3)竞争者角色

竞争者角色以竞争对手的身份出现，与主人公存在相同的目的（最终目的或者阶段目的）的对立角色

竞争者角色的作用：

通过与主人公竞争让玩家感受到“竞争带来的满足感”，使得主人公的成长过程更加直观（即胜利为成长，失败为有待成长）

竞争者角色的创作关键在于“如何刺激玩家的竞争”，给以一种容易引起主人公嫉妒心理或者对抗心理的设定，为此需要给竞争者角色不相上下甚至更强的能力以及相同的目的，并将其描绘成拥有令人厌恶的性格的角色，让玩家“不想输给竞争者”。

给竞争者赋予相同属性也能够引起竞争者之间的激烈斗争

3.2.3与主人公同阵营的定位

即帮助主人公达成目的的定位，使得故事顺利发展下去。另外，这类角色既是帮手，也可以是负担，束缚住主人公的行动。

1)情侣角色

情侣角色是伙伴角色的一个变种，是伙伴角色中与主人公有恋爱关系的角色。

作用：在故事中，情侣角色为主人公带来了动力，其中的恋爱关系也提升了玩家玩游戏的动力；也能为主人公的目的赋予更强的说服力。

处理方法：

  1. 如果给情侣角色增加待攻克对象的定位，即以与情侣角色成为眷属为目的的游戏，即使我们常说的恋爱冒险游戏或者恋爱模拟游戏。
  2. 给情侣角色安排特殊的境遇，也能够提升玩家玩游戏的动力。比如使情侣角色在角色身份或者种族上增加特殊设定，产生“遥不可及”的效果。
  3. 给情侣角色增加特别的内在设定，保守某个秘密或者某个使命，使得玩家激发对情侣角色的兴趣，促使玩家主动关注情侣角色。
  4. 让情侣角色与主人公之间保持特殊的关系。比如儿时的玩伴、寄宿者、或者某个秘密的共有者。

以上这些处理方法既唤起了玩家的兴趣，也为发展恋爱关系提供了障碍，“越是有障碍越有动力克服”，进而提升玩家的动力。

创作要点：

  1. 设计情侣角色时，最好设计一个与之相反的角色，烘托出情侣角色的魅力；
  2. 竞争者也是主人公恋爱关系的良好助燃剂。在竞争角色与情侣角色中设置一些情侣角色与主人公之间不具备的“特殊东西”，竞争关系会更强烈。比如竞争者角色与情侣角色之间是儿时的玩伴、有同样的兴趣爱好、已经订婚等等，激发主人公的嫉妒心。使得主人公通过战胜竞争者，深化与情侣角色之间的感情，使玩家获得更多的成就感。
  3. 竞争角色的出场也可以是主人公对情侣角色产生恋爱感情，通过比较来引出恋爱感情。
  4. 如果游戏一开始就确定主人公与情侣角色之间的恋爱关系，应该给出一个恰当的理由，或者讲一讲二人坠入爱河的经过，使得玩家有足够的情感代入。

情侣角色与主人公之间特殊的感情关系，容易使得情侣角色成为主人公的一个把柄、一个弱点。有时候情侣角色并不一定属于主人公的同一阵营，甚至可能同时有恶人角色的定位。但这时候情侣角色确实有不同寻常的魅力，使之称为“蛇蝎美人”或者“小恶魔角色”。

2)伙伴角色

伙伴角色可以分为以下几种：

  1. 帮助主人公达成目的的战友
  2. 让主人公吐露感情或者意志的倾听者
  3. 让主人公更显眼的衬托者

战友角色：

在游戏剧本中，需求是伙伴角色之母。伙伴角色为主人公提供能力或者条件，提供挑战感。

成为战友角色的必备条件在于拥有主人公不具备的能力或者条件。

作用在于帮助主人公达成目的。

倾听者角色：

作用：

  1. 倾听者角色能够让主人公吐露自己的情感或者想法。使主人公的想法、心情通过对话或行为体现出来，能够加强玩家的理解
  2. 倾听者角色能够给主人公以鼓舞，为主人公提供建议。另外，对话的表现能力比自言自语更强、更加生动。
  3. 主人公与伙伴角色获得同样的体验可以使它们之间的感情获得共鸣，从而使主人公的感情更加细腻地表现出来，增强玩家情感地振幅。

衬托者角色：

  1. 伙伴角色的魅力也是主人公的魅力之一。伙伴角色越加喜欢主人公，主人公就被烘托得更好。
  2. 给伙伴角色赋予相反于主人公的特质，也可以对主人公形成衬托，放大主人公的一些魅力。

创作要点：

  1. 使伙伴角色最初以敌对角色登场。
  2. 并且在成为伙伴前展现出足够的魅力，吊起玩家收服的胃口。
  3. 避免“好好先生”。应该使伙伴角色尽量与主人公保持对立，使之充分发挥作用。伙伴角色学会说“不”，使整个故事有更有趣的插曲。
  4. 不要让伙伴角色成为没有存在感的“死角色”。除了之前介绍到的“战友”、“倾听者”、和“烘托者”三种方法外，还可以通过其他方法使得伙伴角色更加鲜活。

避免“死角色”的方法：

  1. 牺牲：

    主人公身处险境时，伙伴角色舍身相救。使得主人公认识到敌人的强大实力，也使主人公产生强烈的如自责、后悔、愤怒等情感波动。

  2. 变脸：

    突然改变角色定位。比如玩家以为某个角色是伙伴角色，但其实是“叛徒角色”或者“贤者角色”。

给敌人安排伙伴角色使得敌人的存在感提升，使得敌人更有魅力。

让敌人的伙伴帮助主人公，可以使得主人公显得比敌人更有魅力。

3)家人角色：

以任意角色为基准，与该角色有血缘关系或者婚姻关系，认可双方为家人的角色就是家人角色。

好用之处：

  1. 家人角色的定位以“家人就是好的”、“家人应该和好”这一社会基本观念为为前提，能够使玩家产生和好的预期。
  2. 家人角色能够让主人公的情感得到强化。
  3. 将家人角色加入故事，玩家能够自发通过想象理解很多事情，这是其好用之处。

虽然家人角色的存在感在前期很高，但需要通过对话、剧情长期保持存在感。而其它伙伴角色的存在感需要一点点加上去，才能使主人公为其做些什么。
