《游戏剧本怎么写》笔记一

1.主题与故事

1.1游戏剧本创作流程

游戏剧本的创作过程包括两个阶段：分别是策划阶段与剧本制作阶段

策划阶段:确定方向和内容，确定游戏的根本与根基；

剧本制作阶段:遵循策划内容创作，也因此在策划阶段之后

实际上，在实际创作剧本时，“故事”、“角色”、“世界”、“主题”这四个主要部分是同时创作完成的，思维是在这四个成分之间来回跳跃的，它们共同影响最终剧本的完成

1.2主题

一个游戏在立项的时候就应有主题确定出来。无论主题或长或短，都应该有明确的方向，即主题确定整个游戏的方向。
一般来说，在方向感的基础上，主题短句则用词要抽象，长句则要给人吸引力

1.3故事

主题确定方向，进而确定故事是什么样子的
而故事需要传达游戏的主题，以及主角的目的是什么
设定主题，一般要注意以下几点，接下来结合某一项目进行简单说明:

　　1. 游戏目的与主题联系:目的物引发主题（在该项目中游戏的主题是反思，主角的目的是复兴门派，目的与主体之间应该形成这样的循环，即门派覆灭引发主角的反思从而使主角走上复兴门派的道路）；
　　2. 主题足以创作故事:扩充主题，主题与目的拼合

1.4主题与故事

为了使主题与故事之间相符合，应该从主题中导出故事。

方法:从主题引出故事，从主题中发散思维，挖掘故事发展的原因，引出整个故事
材料:（当没有头绪时不妨查查资料）

　　1. 素材库:话题库、创意库记录创意与主题
　　2. 借鉴“原型”
　　3. 使用角色和世界设定，角色创作故事，世界创作故事

1.5表现主题

关于主题的表现形式，需要注意的是“主题不能写在台词里”，应该依靠角色选择与角色行为表现，故事发展表现

2.角色

2.1角色

角色是形象与性质的总体
角色需要推动故事发展
编剧之能在于“能否着重表现角色的目的与意志”（如对师父师叔的回忆，对门派其他人的态度，对师叔的态度，对敌方势力的态度，以及主角与人、人与人之间、主角内心的纠葛），在于“如何表现角色向目标前进”（主角深入敌巢、找回秘宝、带领大家复兴门派）。前者与后者的区别在于，前者主要是内部情感、态度来表现，后者以外部的行为来表现。
纠葛，强化角色之间的纠葛（即人与人的对立，或内心的纠结），是一种体现角色意志、目的、和感情的手段。纠葛包括战斗对立冲突等“外在纠葛”与内心的“内在纠葛”
体现角色的魅力与原则，能够让玩家产生共鸣
创作角色三个重点：

　　1. 角色有魅力有个性（角色设定问题）
　　2. 在不同角色定位上多花心思，避免角色重复（角色定位问题）

   3. 角色的目的意志感情和行动<u>与故事相联系</u>（角色与故事关联性问题）

      接下来将从角色的设定、定位、关联性这几方面进行总结。

2.2角色设定

角色设定表是帮助设计角色形象的重要工具，其中主要涉及以下几个项目：
名字（名字决定角色印象，其中主要的因素是发音，名字中有g或d发音的角色往往给人坏人印象，l对应诱惑或怪异，n对应温柔可爱，p有些俏皮，t更像是主角；但我认为这只在国外或者音译作品中有体现，在国内个人更倾向于主角名字的易读性）、年龄（能够表明角色之间的关系，年龄本身能让玩家联系到某种印象）、职业（职业带有明显的先入为主的印象，或者可以制造反差）、相貌（主要写与故事相关的特征）、头发（体现角色内在性格、外在个性，或者状态变化）、体格（体格反应个人年龄、状态、生活习惯与性格）、服装（帮助把握角色详细信息，如兴趣和思维方式）、性格（能够表示角色情感或行为的倾向，考虑性格的成因，让台词更有深度更贴合角色）、特技（凸显不同的特性，可以用以实现目标的有效手段，不一定要是攻击技能）、弱点（让玩家更容易代入角色，唤起共鸣，人物鲜活）、口吻（直接体现角色个性，增加口头禅）、剧本设定（角色目的、经历、状况以及必要信息等）
在性格的设计中，“定位型性格”是一个比较受重视的手法，所谓“定位型性格”即游戏中常见“哥哥那样的性格”这之类的说法。定位型性格的用处在于：

　　1. 把身份投射到角色性格上，使人际关系更加直观
　　2. 定位型性格的变化阐明角色立场及变化
3. 利用定位型性格让角色设定更有说服力

     值得注意的是，因为定位型性格中有刻板印象的作用，在实际应用的过程中，应该避免刻板、尝试增加错位反差的设定或性格

       另外，角色设定表要准确传达信息，避免想当然

3.角色定位

3.1角色的定位

定位是分配给故事中各个角色的立场，可以明确角色之间关系，清楚掌握角色行动，避免角色重复
外观相同、定位不同的角色之间，有明显区别；但外观不同，定位相同的角色之间会因为外观特征而有印象的顶替。

3.2角色的5种定位：

　　1. 与游戏系统直接关联的定位：
1.1 主人公
　　 1.2 待攻克对象
　　 1.3 系统角色

       　　2. 与主人公**对立**的定位：（推动剧本发展）
   2.1 敌人角色
          　　        2.1.1 恶人角色
          　　        2.1.2 敌对角色
          　　    2.2 难关角色
     　　    2.3 竞争者角色

      3. 与主人公**同阵营**的定位：（推动剧本发展）
            3.1 情侣角色
       　　    3.2 伙伴角色
         　　    3.3 家人角色

               4. 给故事**带来变化**的定位：（辅助剧本发展）
              4.1 契机角色
         　　    4.2 救星角色
           　　    4.3 叛徒角色
       　　    4.4 贤者角色

                 5. **补充和强化故事**的定位：（辅助剧本发展）
                5.1 缓冲角色
                　　    5.2 深论角色
                　　    5.3 动物角色
                　　        5.3.1 “妹妹”角色
                　　    5.4 闲杂角色
                　　**一个角色可以拥有多个定位**，角色越重要，拥有定位越多
                　　且故事发展可推动定位改变

                接下来对这几种分类分别说明。

3.2与游戏系统直接关联的定位

3.2.1主人公

主人公作为玩家的分身，是玩家容易带入感情的角色
为了实现这一点，主人公的描绘方式在于：

　　1. 主人公的（外观）形象要以没有太多特征的方式描绘，应该感觉不到太多个性
　　2. 允许玩家自定义主人公的名字
　　3. 尽量不让主人公有过多台词
　　4. 不能强制玩家接受主人公的个性，给以容易让大多人接受的个性，如优柔寡断
